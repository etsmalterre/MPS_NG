## Production Process

The manufacturing flow follows this chain:

```
Fil (Yarn) → Tricotage (Knitting) → Écru (Raw fabric) → Teinture (Dyeing) → Fini (Finished) → Confection (Assembly) → Expédition (Shipping)
```
